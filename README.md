# 퇴사 시뮬레이터 🔥

직장인 스트레스 해소용 미니게임 2종. 의존성 없는 단일 `index.html` — 웹/모바일 겸용.

## 게임

### 😤 사직서(무기) 던지기
- 사직서를 드래그해 던지거나, **부장님을 터치하면 장착 무기가 얼굴로 자동 발사**됩니다
- 던진 속도에 따라 톡 → 퍽! → **크리티컬!!** 3단계 연출 (화면 진동, 파티클, 합성 타격음, 모바일 진동)
- 콤보 시스템 + 부장님 멘탈 게이지. 멘탈이 바닥나면 K.O. → 다음 부장님이 무한 출근 (세대마다 체력 +30)
- 타격/KO마다 🥤 사이다를 벌어 **사내 매점**에서 무기 구매 — 무기를 사면 투사체가 그 무기로 바뀝니다
  - 🖊️ 결재 볼펜: 다트처럼 직선으로 날아가 얼굴에 **꽂힘**
  - ⌨️ 기계식 키보드: 하늘에서 **내리찍기**
  - 🔫 고무줄 권총: **즉발 사격** (트레이서 + 총성)
  - 🏏 몽둥이 / 🪑 의자 / 💼 서류가방: 회전 투척

### 💥 회사 폭파 (추후 별도 개발 예정 — 현재 비활성화)
- 코드는 남아 있으나 탭이 숨겨져 있습니다. 전국 실시간 집계는 [counterapi.dev](https://counterapi.dev) 공용 카운터 기반.

## 실행

정적 파일이라 아무 웹서버로 열면 됩니다:

```sh
npx serve .
```

사운드는 전부 Web Audio API 합성이라 오디오 파일이 없고, 캐릭터는 인라인 SVG입니다.
