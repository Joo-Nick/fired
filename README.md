# 퇴사 시뮬레이터 🔥

직장인 스트레스 해소용 미니게임 2종. 의존성 없는 단일 `index.html` — 웹/모바일 겸용.

## 게임

### 😤 사직서 던지기
- 사직서를 드래그해서 부장님 얼굴에 던지세요. 어느 방향에서 던져도 맞습니다.
- 던진 속도에 따라 톡 → 퍽! → **크리티컬!!** 3단계 연출 (화면 진동, 파티클, 합성 타격음, 모바일 진동)
- 콤보 시스템 + 부장님 멘탈 게이지. 멘탈이 바닥나면 K.O. → 다음 부장님이 무한 출근

### 💥 회사 폭파
- 빨간 퇴사 버튼을 누르면 (주)야근 빌딩이 폭발하고 1.4초 뒤 자동 재건축
- **전국 실시간 집계**: 오늘 전국에서 몇 명이 회사를 몇 번 파괴했는지 실제로 카운트 ([counterapi.dev](https://counterapi.dev) 공용 카운터, 한국 시간 기준 일별 리셋)
- 내 파괴 횟수는 localStorage에 누적, 10/50/100/200/500회 칭호 달성

## 실행

정적 파일이라 아무 웹서버로 열면 됩니다:

```sh
npx serve .
```

사운드는 전부 Web Audio API 합성이라 오디오 파일이 없고, 캐릭터는 인라인 SVG입니다.
